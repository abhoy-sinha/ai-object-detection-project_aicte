# 🎯 AI Object Detection
**Internship Project — AICTE | IBMSKILLBUILDS | Edunet Foundation**

## 📌 Project Overview

This project is a **real-time AI Object Detection system** that runs directly in a web browser using a live camera feed. It detects and identifies everyday objects instantly and displays bounding boxes with confidence scores. The application is lightweight, fast, and works without installing heavy software or hardware.

---

## 🚀 Features

* Real-time object detection using webcam
* Bounding box visualization
* Confidence score display
* Works directly in browser
* Lightweight and responsive UI
* Adjustable FPS control
* Toggle AI detection ON/OFF

---

## 🧠 Problem Statement

Traditional software cannot automatically interpret visual data from live video streams. There is a need for a system that can intelligently detect and recognize objects in real time without manual input, installation, or high-end hardware.

---

## 💡 Proposed Solution

A browser-based AI system that uses a trained object detection model to analyze live video frames and identify objects instantly. The system processes frames continuously and overlays detection results visually.

---

## 🛠 Technology Used

* HTML5
* CSS3
* JavaScript
* TensorFlow.js
* COCO-SSD Model
* Web Camera API

---

## ⚙️ Working Principle

1. Camera captures live video frames
2. Frames sent to detection model
3. AI predicts objects in frame
4. Bounding boxes drawn on canvas
5. Labels + confidence displayed

---

## 📊 Results

* Successfully detects real-world objects in real time
* Identifies common objects such as person, cup, mouse, scissors
* Runs smoothly across devices and browsers
